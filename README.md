# Create your amazing placeholder

[See a demo.](https://happyjayxin.github.io/ColorPlaceholder/)

## Usage

### html
```html
<input type="text" id="name">
```
### css
```css
#name{
    width: 200px;
}
```
### jquery ( 網頁請先載入[jquery](https://jquery.com/) )
```js
$('#name').ColorPlaceholder();


$('#name').ColorPlaceholder({
   text:'快點告訴我你的名子'
});


$('#name').ColorPlaceholder({
   color:'#45aaf2',
   text:'快點告訴我你的名子'
});
```
